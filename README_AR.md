# Discord AI Bot - بوت ديسكورد ذكي

بوت ديسكورد متكامل يجمع بين قوة الإدارة والذكاء الاصطناعي المتقدم لتحسين تجربة خادمك.

## المميزات الرئيسية

### 🤖 الذكاء الاصطناعي المتقدم
يعتمد البوت على نموذج **Google Gemini** الحديث للإجابة على الأسئلة والمساعدة في حل المشاكل بذكاء عالي. يمكن للمستخدمين طرح أسئلة متنوعة والحصول على إجابات دقيقة وسريعة.

### 🛡️ أوامر إدارية قوية
يوفر البوت مجموعة شاملة من أوامر الإدارة تشمل:
- **!kick @user [reason]** - طرد عضو من السيرفر
- **!ban @user [reason]** - حظر عضو من السيرفر
- **!warn @user [reason]** - تحذير عضو
- **!mute @user [reason]** - كتم صوت عضو لمدة ساعة
- **!unmute @user** - فتح صوت عضو

### 📊 لوحة تحكم احترافية
لوحة تحكم ويب متطورة توفر:
- عرض إحصائيات البوت والخوادم
- إدارة إعدادات الخوادم
- مراقبة السجلات والأحداث
- تخصيص إعدادات الذكاء الاصطناعي

### 📝 نظام تسجيل شامل
يسجل البوت جميع الأوامر والأحداث والأخطاء في قاعدة البيانات لتتبع كامل نشاط السيرفر.

## الأوامر المتاحة

### الأوامر العامة

| الأمر | الوصف |
|------|-------|
| `!ping` | عرض تأخير البوت (Latency) |
| `!help` | عرض قائمة الأوامر المتاحة |
| `!ai [question]` | اسأل الذكاء الاصطناعي سؤال |

## البنية التقنية

### المكونات الأساسية

**Backend:**
- **Node.js + Express** - خادم الويب الرئيسي
- **tRPC** - نظام الاتصال بين الخادم والعميل
- **MySQL/TiDB** - قاعدة البيانات
- **Drizzle ORM** - إدارة قاعدة البيانات
- **discord.js** - مكتبة بوت الديسكورد
- **Google Generative AI** - API الذكاء الاصطناعي

**Frontend:**
- **React 19** - مكتبة واجهة المستخدم
- **Tailwind CSS 4** - تصميم الواجهة
- **Recharts** - الرسوم البيانية
- **shadcn/ui** - مكونات واجهة المستخدم

### هيكل المشروع

```
discord-ai-bot/
├── client/                 # تطبيق الويب (React)

```

## التثبيت والإعداد

### المتطلبات
- Node.js 18+
- npm أو pnpm
- حساب Google Cloud مع تفعيل Gemini API
- توكن بوت Discord

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd discord-ai-bot
```

2. **تثبيت المكتبات**
```bash
pnpm install
```

3. **إعداد متغيرات البيئة**
أنشئ ملف `.env` في جذر المشروع:
```env
DATABASE_URL=mysql://user:password@host:port/database
DISCORD_TOKEN=your_discord_bot_token
GEMINI_API_KEY=your_gemini_api_key
JWT_SECRET=your_jwt_secret
VITE_APP_ID=your_oauth_app_id
OAUTH_SERVER_URL=https://api.manus.im
```

4. **إعداد قاعدة البيانات**
```bash
pnpm db:push
```

5. **تشغيل المشروع**
```bash
# في بيئة التطوير
pnpm dev

# البناء للإنتاج
pnpm build
pnpm start
```

### السجلات
```
GET /api/trpc/bot.getRecentLogs?serverId=xxx&limit=50
GET /api/trpc/bot.getModerationLogs?serverId=xxx&limit=50
GET /api/trpc/bot.getAIConversations?serverId=xxx&limit=50
```

## قاعدة البيانات

### الجداول الرئيسية

**discord_servers** - معلومات الخوادم
- serverId: معرف الخادم
- serverName: اسم الخادم
- prefix: بادئة الأوامر
- aiEnabled: تفعيل الذكاء الاصطناعي

**command_logs** - سجل الأوامر
- commandName: اسم الأمر
- userId: معرف المستخدم
- success: هل نجح الأمر
- errorMessage: رسالة الخطأ إن وجدت

**ai_conversations** - محادثات الذكاء الاصطناعي
- userMessage: رسالة المستخدم
- aiResponse: رد الذكاء الاصطناعي

**moderation_logs** - سجل الإجراءات الإدارية
- action: نوع الإجراء (kick, ban, warn)
- reason: السبب

## استكشاف الأخطاء

### البوت لا يستجيب
1. تحقق من أن توكن Discord صحيح
2. تأكد من أن البوت لديه الصلاحيات المطلوبة
3. تحقق من السجلات في الكونسول

### الذكاء الاصطناعي لا يعمل
1. تحقق من أن مفتاح Gemini API صحيح
2. تأكد من تفعيل Gemini API في Google Cloud
3. تحقق من حد الطلبات (Rate Limit)

### مشاكل قاعدة البيانات
1. تحقق من سلسلة الاتصال (DATABASE_URL)
2. تأكد من أن قاعدة البيانات تعمل
3. جرب تشغيل `pnpm db:push` مجددا

## التطوير والمساهمة

### إضافة أمر جديد
1. أنشئ ملف جديد في `server/bot/commands/`
2. اتبع نمط الأوامر الموجودة
3. أضفه إلى قائمة الأوامر في `commandHandler.ts`

### إضافة API endpoint جديد
1. أضفه إلى `server/routers/bot.ts`
2. استخدمه في الواجهة الأمامية مع `trpc`
